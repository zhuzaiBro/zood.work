# 面试题批量导入管理页面

## 🎯 功能说明

这是一个可视化的面试题批量导入工具，支持通过上传 CSV 文件快速导入大量面试题到指定题集。

## 📍 访问路径

- **URL**: `/admin/questions`
- **入口**: 用户头像下拉菜单 → "面试题管理"

## ✨ 主要功能

### 1. 选择目标题集
- 从下拉列表选择要导入的目标题集
- 自动加载所有可用的题集

### 2. 上传 CSV 文件
- 支持拖拽或点击上传
- 自动验证文件格式（必须是 .csv 文件）
- 显示文件大小

### 3. 解析预览
- 点击"解析 CSV"按钮，自动解析文件内容
- 预览前 10 条数据
- 显示解析结果统计

### 4. 批量导入
- 点击"开始导入"按钮，逐条导入数据
- 实时显示导入进度
- 显示成功和失败数量
- 详细的错误信息

### 5. 下载模板
- 点击"下载模板"按钮获取标准 CSV 模板
- 模板包含示例数据和正确格式

## 📋 CSV 文件格式要求

### 必需字段

```csv
title,content,is_vip,difficulty
"问题标题","答案内容",false,简单
```

### 完整格式（包含可选字段）

```csv
title,content,is_vip,difficulty,collection_id,vip_level_required,sort
"问题标题","答案内容",false,简单,,,1
```

### 字段说明

| 字段 | 必填 | 类型 | 说明 | 示例 |
|------|------|------|------|------|
| title | ✅ | string | 问题标题 | "什么是闭包?" |
| content | ✅ | string | 答案内容（支持 Markdown） | "闭包是指..." |
| is_vip | ✅ | boolean | VIP 内容 | true / false |
| difficulty | ✅ | string | 难度 | 简单/中等/困难 |
| collection_id | ⭕ | UUID | 题集 ID（页面会自动填充） | - |
| vip_level_required | ⭕ | number | VIP 等级 | 1-5 |
| sort | ⭕ | number | 排序号 | 1, 2, 3... |

**注意**: `collection_id` 字段在 CSV 中可以忽略，页面会使用你选择的题集 ID。

## 🎨 界面说明

### 操作区域
1. **选择题集下拉框** - 选择目标题集
2. **文件上传输入框** - 选择 CSV 文件
3. **下载模板按钮** - 获取标准模板
4. **解析 CSV 按钮** - 解析并预览数据
5. **开始导入按钮** - 执行批量导入

### 进度显示
- 导入时显示进度条
- 实时更新当前进度（如：45 / 100）

### 提示信息
- **成功提示**（绿色）: 操作成功的信息
- **错误提示**（红色）: 失败原因和错误详情

### 数据预览
- 显示解析后的前 10 条数据
- 每条数据显示：
  - 序号和标题
  - VIP 标签（如果是 VIP 内容）
  - 难度标签（简单/中等/困难）
  - 答案内容预览（前 150 字符）

## 🚀 使用流程

### 方式一：使用模板（推荐新手）

```
1. 点击"下载模板"按钮
   ↓
2. 使用 Excel/Google Sheets 打开模板
   ↓
3. 参考示例数据，填写你的题目
   ↓
4. 保存为 CSV 格式（UTF-8 编码）
   ↓
5. 在页面选择目标题集
   ↓
6. 上传 CSV 文件
   ↓
7. 点击"解析 CSV"预览数据
   ↓
8. 确认无误后点击"开始导入"
```

### 方式二：自己创建 CSV

```
1. 创建 CSV 文件，添加表头：
   title,content,is_vip,difficulty,vip_level_required,sort
   ↓
2. 填写数据（注意格式要求）
   ↓
3. 在页面选择目标题集
   ↓
4. 上传文件并导入
```

## 💡 使用技巧

### 1. Excel 编辑多行内容
- 在单元格内按 `Alt + Enter`（Windows）或 `Option + Enter`（Mac）换行
- Excel 会自动为多行内容添加双引号

### 2. Markdown 格式支持
内容支持完整的 Markdown 语法：

```markdown
# 标题
**加粗文本**
- 列表项

\`\`\`javascript
// 代码块
function example() {}
\`\`\`
```

### 3. 批量导入建议
- **小规模测试**: 先导入 5-10 条数据测试
- **分批处理**: 超过 500 条建议分多次导入
- **备份数据**: 导入前保存原始 CSV 文件

### 4. 错误处理
如果部分数据导入失败：
1. 查看错误信息，了解失败原因
2. 修正 CSV 文件中的错误数据
3. 重新导入（成功的不会重复）

## ⚠️ 注意事项

### 1. 文件编码
- 必须使用 **UTF-8** 编码
- Excel 另存为时选择"CSV UTF-8（逗号分隔）"

### 2. 特殊字符处理

**包含逗号的内容**
```csv
title,content
"标题","内容包含,逗号"
```

**包含双引号的内容**
```csv
title,content
"标题","内容中的""双引号""要双写"
```

**包含换行的内容**
```csv
title,content
"标题","第一行
第二行
第三行"
```

### 3. 字段验证
- `is_vip` 必须是 `true` 或 `false`（小写）
- `difficulty` 建议使用中文：简单、中等、困难
- `vip_level_required` 必须是 1-5 的数字

### 4. 数据安全
- 页面需要登录才能访问
- 建议添加管理员权限控制（待实现）
- 导入前会验证所有数据格式

## 🐛 常见问题

### Q1: 上传后提示"解析失败"
**A**: 检查以下几点：
- 文件编码是否为 UTF-8
- 表头是否完整正确
- 数据格式是否符合要求

### Q2: 部分数据导入失败
**A**: 查看错误详情，常见原因：
- 必填字段为空
- `is_vip` 不是 true/false
- `vip_level_required` 超出范围

### Q3: Markdown 格式显示不正确
**A**: 确保：
- 代码块使用三个反引号包裹
- 整个 content 字段用双引号包裹
- 没有未闭合的标签

### Q4: 选择题集后无法上传
**A**: 可能原因：
- 浏览器缓存问题，刷新页面
- 检查网络连接
- 查看浏览器控制台错误信息
